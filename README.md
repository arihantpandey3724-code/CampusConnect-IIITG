This is a simple Android mobile app designed to be easy to use and visually clean. The app has a smooth layout, clear buttons, and a modern look so that users can easily understand and navigate it. It works well on different Android phones and provides a comfortable user experience.
This project was created mainly for learning and practice purposes. It shows how a mobile app can be designed in an organized and structured way, making it easy to improve or add new features in the future. The app is suitable for demonstration, academic projects, and basic user testing.
